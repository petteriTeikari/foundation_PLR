## Document Types

### Implementation Plans

Documents describing how to implement features:

```markdown
# Feature: [Name]

## Background
Why this feature is needed.

## Approach
How to implement it.

## Tasks
1. [ ] Task 1
2. [ ] Task 2
```

### Decision Records

Documents recording technical decisions:

```markdown
# Decision: [Title]

## Status
Accepted/Proposed/Deprecated

## Context
What prompted this decision?

## Decision
What we decided.

## Consequences
What results from this decision.
```

### Lessons Learned

Documents capturing what worked and what didn't:

```markdown
# Meta-Learning: [Topic]

## What Happened
Description of the situation.

## What Worked
Successful approaches.

## What Didn't Work
Failed approaches and why.

## Recommendations
What to do differently next time.
```
